I: Commands

There are over 10 commands in CoreOS v1.0

1: $pwd: Shows where you are

2: $ping: Pong!

3: help: Shows the commands list or '?'

4: $calcu: starts coreos calculator

5: $code: shows the code of coreos

6: $matrix: 1010100101010101
